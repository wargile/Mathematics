Synthetic division:
--------------------------------------------------------------------------------------------

(2x^3 – x + 4) / (x + 3)
That is:
2x^3 + 0x^2 – 1x + 4
Then take out all the coefficients:

-3 _| 2  0  -1   4
        -6  18 -51  
------------------
      2 -6  17 -47 (-47 is then the remainder)

Which gives: 2x^2 - 6x + 17 + (-47/(x + 3))
-------------------------------------------

(4x^3 – 2x^2 – 10x + 1) / (x – 2)
2 _| 4  -2 -10   1
         8  12   4  
------------------
     4   6   2   5

Which gives: 4x^2 + 6x + 2 + (5/(x - 2))

Adding algebraic fractions:
--------------------------------------------------------------------------------------------
Find the least common denominator (LCD):
(7 / 3xy^2) + (5 / 2x^2y)
The LCD is 6x^2y^2
Divide the LCD with the existing denominators:
6x^2y^2 / 3xy^2 = 2x
6x^2y^2 / 2x^2y = 3y
Multiply the numerators with these values, and you get:
((7 * 2x) / (6x^2y^2)) + ((5 * 3y) / (6x^2y^2))
Answer: (14x + 15y) / (6x^2y^2)


--------------------------------------------------------------------------------------------
Subtracting algebraic fractions:
--------------------------------------------------------------------------------------------
Find the least common denominator (LCD):
(5x/4y) - (2y/5x)
The LCD is 20xy
Divide the LCD with the existing denominators:
20xy / 4y = 5x
20xy/ 5x = 4y
Multiply the numerators with these values, and you get:
((5x * 5x)/20xy) - ((2y * 4y)/20xy)
Answer: (25x^2 - 8y^2) / (20xy)

Using the quadratic formula:
--------------------------------------------------------------------------------------------
The quadratic formula (MEMORIZe IT!) is:
x = (-b +/- sqrt(b^2 - 4ac)) / 2a
It is good for solving equations that are not easily factorable.

Try: 4x^2 + 16x + 15 = 0
That means: a = 4, b = 16, c = 15
Substitute:
x = (-16 +/- sqrt(16^2 - (4*4*15))) / (2*4)
x = (-16 +/- sqrt(256 - 240)) / 8
x = (-16 +/- sqrt(16)) / 8
x = (-16 +/- 4) / 8
x = -12/8 or x = -20/8
Answer: x = -3/2 or x = -5/2 (x = -1.5 or x = -2.5) (x = -1and1/2 or x = -2and1/2)

Try: 7x^2 + 25x + 12 = 0
That means: a = 7, b = 25, c = 12
Substitute:
x = (-25 +/- sqrt(25^2 - (4*7*12))) / (2*7)
x = (-25 +/- sqrt(625 - 336)) / 14
x = (-25 +/- sqrt(289)) / 14
x = (-25 +/- 17) / 14
Answer: x = -4/7 or x = -3
